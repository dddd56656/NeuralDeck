## 📖 简介 (Introduction)

**NeuralDeck** 是一款基于 Flutter 构建的**离线多模态资产编译终端**。

它不仅仅是一个卡片生成器，而是一个运行在用户手掌中的**AGI（通用人工智能）节点**。利用端侧 AI 技术，它能通过视觉（OCR）、听觉（ASR）感知物理世界，利用本地大模型（LLM）进行重构，并最终以具有自我意识的 Live2D 虚拟生命形态呈现。

**核心特性：** 0 延迟、0 流量消耗、100% 数据隐私安全。

## ⚡ 核心模块 (Core Modules)

### 1. 👁️ 视觉义眼 (Vision Circuit)
* **技术栈:** Google ML Kit (On-Device)
* **功能:** 毫秒级 OCR 光学字符识别 + 离线神经机器翻译。
* **能力:** 能够读取实体卡牌、商品包装上的日文/英文信息，并转化为结构化数据。

### 2. 🧠 认知核心 (Cognitive Core)
* **技术栈:** Llama-3 (GGUF Quantized) + SQLite Vector
* **功能:** 本地 RAG（检索增强生成）与逻辑推理。
* **能力:**
    * **生成模式:** 基于残缺信息补全设定（如：自动补全怪兽背景故事）。
    * **诊断模式:** 作为“AI 鉴定师”，评估资产价值、战力评级与兼容性分析。

### 3. 💃 神经同步 (Neural Sync)
* **技术栈:** Live2D Cubism + Flutter TTS
* **功能:** 2.5D 全息投影与语音合成。
* **能力:** 生成的数字资产拥有“肉体”，能跟随手指视线移动，并能用语音播报自身属性。

### 4. 🦾 具身感知 (Embodied Sense)
* **技术栈:** Sensors Plus + Haptics
* **功能:** 物理引擎反馈。
* **能力:**
    * **前庭觉:** 倾斜手机，卡片全息光效随重力改变。
    * **触觉:** 基于诊断评级（S/A/B）反馈不同强度的震动波。
